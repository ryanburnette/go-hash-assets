## Usage

```
hash-assets -dir <directory> [-d] [-hash-len] [-asset-exts] [-markup-exts]
```

```
Usage:
	-asset-ext string
		Asset file extensions (default ".css,.js")
	-d	Dry run
	-dir string
		Working directory path, required
	-hash-len string
		Hash string length (default "10")
```
